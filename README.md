# The Fraud Desk —《反诈柜台》

> 一款"反诈版 Papers, Please"。你是夜班反诈分析员,逐条审查来路不明的短信 / 转账 / 来电,
> 靠识别破绽决定 **SAFE(放行)** 还是 **SCAM(拦截)**,救下差点被骗的人。

为 **[The Demo Jam 2026](https://itch.io/jam/the-demo)**(截止 2026-07-07)制作的可玩切片(vertical slice)。

---

## 这是什么 / 不是什么

- **MVP(本仓库)**:1 个夜班、5 个案件、放行/拦截二选一、止损计分、逐案揭晓红旗信号、结算评级。
- **完整愿景(游戏内结算页有展示)**:8 个班次,覆盖老年人 / 学生 / 小微企业目标;20+ 真实诈骗剧本(深伪语音、收款码掉包、假客服…);随玩法自适应的"职业生涯"模式;以及一个**可免费分享给家人的开放案例库**。
- **公益使命**:把"反诈意识"练成肌肉记忆——免费、人人可用。

## 运行

纯静态单文件,零依赖、秒开。

```bash
# 任选其一,在本目录下:
python -m http.server 5273
# 然后浏览器打开 http://localhost:5273
```

或直接双击 `index.html` 也能跑(无需服务器)。

## 文件

- `index.html` — 整个游戏(HTML + CSS + JS 全内联,方便 itch.io 单 zip 上传)。
- 案件数据在 `<script>` 顶部的 `CASES` 数组里,**数据驱动**——加新剧本只要往数组里加一条,无需改逻辑。这就是"扩到 20+ 剧本"的接口。

## 双语(中 / EN)

- 顶部 `EN / 中` 开关一键切换,**实时重渲染**当前画面,分数不丢、不重复计分。
- 首次打开按浏览器语言**自动选择**(`navigator.language`),之后记在 `localStorage`。
- 所有玩家可见文案都在 `<script>` 顶部:UI 串在 `L = { en, zh }`,案件内容在 `CASES`(每个字段都是 `{ en, zh }`)。加语种只要再加一个 key;货币符号随语言 `$ ↔ ¥`。

## 设计决策记录

- **二选一(SAFE/SCAM)而非三键(放行/拦截/上报)**:MVP 阶段砍掉"上报",让核心循环更紧、计分更清晰。完整版可把"上报反诈中心"作为高阶动作回归。
- **止损金额做计分**:比单纯对错更有情绪张力——玩家直观看到"救回了多少钱 / 害人损失了多少"。
- **数据驱动 + i18n**:案件是 `CASES` 数组,文案是 `L` 字典,逻辑与内容彻底分离——这是同时扩"20+ 剧本"和"多语种"的接口。

## 用 Claude Code 接着 vibe 的下一步建议

1. **加案件**:往 `CASES` 加 3-5 条,凑成一个 8-10 案的班次(深伪语音、收款码掉包、假"公检法"、中奖、冒充客服)。每条记得中英都写。
2. **手感**:决策加个倒计时条 → 紧张感;音效(打字声、对错提示)。
3. **分享钩子**:结算页加"把这个案子转给家人"按钮,呼应公益使命。
4. **(可选)第三语种**:`L` 里加一个 key 即可,架构已就绪。
