# The Fraud Desk —《反诈柜台》

> 一款"反诈版 Papers, Please"。你是夜班反诈分析员,逐条审查来路不明的短信 / 转账 / 来电,
> 靠识别破绽决定 **SAFE(放行)** 还是 **SCAM(拦截)**,救下差点被骗的人。

为 **[The Demo Jam 2026](https://itch.io/jam/the-demo)**(截止 2026-07-07)制作的可玩切片(vertical slice)。

---

## 这是什么 / 不是什么

- **MVP(本仓库)**:**案件库 16 桩**,每个夜班从库里**随机抽取 5–10 桩**;每桩**限时 25 秒决策**(超时算漏判、诈骗款照丢);放行/拦截二选一、止损计分、逐案揭晓红旗信号、结算评级。
- **案件取材**:中国高发电诈类型(刷单返利、冒充公检法、注销校园贷/虚假征信、AI 换脸冒充领导、机票退改签、取消自动扣费、虚假理赔、中奖等),改编自国家反诈中心《2025 版宣传手册》及公开案例;另含 5 桩"真消息"用于训练不误判。
- **完整愿景(游戏内结算页有展示)**:8 个班次,覆盖老年人 / 学生 / 小微企业目标;60+ 真实诈骗剧本;随玩法自适应的"职业生涯"模式;以及一个**可免费分享给家人的开放案例库**。
- **公益使命**:把"反诈意识"练成肌肉记忆——免费、人人可用。

## 文件

- `index.html` — 整个游戏(HTML + CSS + JS 全内联,方便 itch.io 单 zip 上传)。
- 案件库在 `<script>` 顶部的 `LIBRARY` 数组里,**数据驱动**——加新剧本只要往数组里加一条(中英各一套),无需改逻辑。每个班次由 `drawRound()` 随机抽取并保证至少各含一桩"诈/真"。
- 决策倒计时秒数由常量 `DECISION_SECONDS`(默认 25)控制。

## 设计 / Design

"暖纸调案件档案(Case File Dossier)"——把游戏包装成一本**互动调查档案**,走高端编辑/杂志风:

- 字体:**Fraunces**(高对比衬线,大标题/抽引语/首字下沉) + **Inter**(正文/UI 眉题),系统衬线兜底。
- 配色:暖奶油纸 `#f3ede1` + 暖墨黑 `#1c1813` + 图章红 `#a72b22` + 核验绿 `#2f6b4f` + 物证赭 `#9c7320`。
- 编辑工艺:双线报头、眉题 kicker、案件编号(`CASE 01 / 05`)、物证标签(`EXHIBIT A — TRANSCRIPT`)、首字下沉、橡皮**图章判定**(BLOCKED/CLEARED/MISSED)、抽引语点题、版权页 colophon、Müller-Brockmann 式细线网格。
- 每个案件有编辑式标题(如《三十分钟冻结术》),中英各一套。

## 双语(中 / EN)

- 顶部 `EN / 中` 开关一键切换,**实时重渲染**当前画面,分数不丢、不重复计分。
- 首次打开按浏览器语言**自动选择**(`navigator.language`),之后记在 `localStorage`。
- 所有玩家可见文案都在 `<script>` 顶部:UI 串在 `L = { en, zh }`,案件内容在 `CASES`(每个字段都是 `{ en, zh }`)。加语种只要再加一个 key;货币符号随语言 `$ ↔ ¥`。

## 设计决策记录

- **二选一(SAFE/SCAM)而非三键(放行/拦截/上报)**:MVP 阶段砍掉"上报",让核心循环更紧、计分更清晰。完整版可把"上报反诈中心"作为高阶动作回归。
- **止损金额做计分**:比单纯对错更有情绪张力——玩家直观看到"救回了多少钱 / 害人损失了多少"。
- **数据驱动 + i18n**:案件是 `CASES` 数组,文案是 `L` 字典,逻辑与内容彻底分离——这是同时扩"20+ 剧本"和"多语种"的接口。

## 用 Claude Code 接着 vibe 的下一步建议

1. **继续扩库**:往 `LIBRARY` 加更多剧本(裸聊敲诈、网游交易、虚假贷款、保健品坑老…),库越大随机性越强。每条中英都写。
2. **手感/反馈**:加音效(打字声、对错提示);倒计时最后 5 秒可加心跳/闪烁;连对 streak 加分。
3. **难度曲线**:按 `DECISION_SECONDS` 随班次递减做"加速档";或分老年/学生/企业专题班次。
4. **分享钩子**:结算页加"把这个案子转给家人"按钮,呼应公益使命。
5. **(可选)第三语种**:`L` 里加一个 key 即可,架构已就绪。
